    Keypair

    `Keypair` represents public (and secret) keys of the account. Currently `Keypair` only supports ed25519 but in a future this class can be abstraction layer for other public-key signature systems. Use more convenient methods to create `Keypair` object: * `Keypair.fromPublicKey` * `Keypair.fromSecret` * `Keypair.random`
