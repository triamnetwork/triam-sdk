    Operation

    new Operation()

    `Operation` class represents [operations](https://www.stellar.org/developers/learn/concepts/operations.html) in Stellar network. Use one of static methods to create operations: * `Operation.createAccount` * `Operation.payment` * `Operation.pathPayment` * `Operation.manageOffer` * `Operation.createPassiveOffer` * `Operation.setOptions` * `Operation.changeTrust` * `Operation.allowTrust` * `Operation.accountMerge` * `Operation.inflation` * `Operation.manageData` * `Operation.createAsset"` * `Operation.changeAsset` * `Operation.limitAsset`
